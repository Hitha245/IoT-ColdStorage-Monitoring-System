# IoT-Based Real-Time Cold Storage Monitoring System

## Overview

An IoT-enabled cold storage monitoring system developed using ESP32 and Python to continuously monitor environmental conditions inside refrigerated storage facilities.

The system collects real-time sensor data, uploads it to the cloud, generates alerts when predefined thresholds are exceeded, and supports remote monitoring.

---

## Features

- Real-time Temperature Monitoring
- Humidity Monitoring
- Gas Leakage Detection
- Moisture Detection
- Door Open Detection
- GPS Location Tracking
- GSM SMS Alerts
- Relay Based Cooling Control
- Blynk Cloud Integration
- CSV Data Logging
- Python Monitoring Dashboard
